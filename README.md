# Автотесты для Кинопоиска

## Описание проекта

В работе приложение “Кинопоиск” - крупнейший русскоязычный интернет-сервис о кино и онлайн-кинотеатр.
Он объединяет обширную базу данных фильмов, сериалов и персон, рейтинги, отзывы, а также возможность легального просмотра контента по подписке.

Цель проекта: Разработать набор автотестов для проверки ключевого функционала онлайн-кинотеатра «Кинопоиск» (https://www.kinopoisk.ru/) и его публичного API (https://api.poiskkino.dev)

Проект содержит автоматизированные тесты (UI и API) для онлайн-кинотеатра Кинопоиск.

Тесты UI периодически просят пройти капчу - она вводится вручную, без автоматизации.

### Структура проекта
kinopoisk_autotests/

├── config/ # Конфигурации и тестовые данные

│ ├── settings.py # Настройки (URL, токены, селекторы)

│ └── test_data.json # Тестовые данные

├── pages/ # PageObject для UI тестов

│ ├── base_page.py # Базовый класс страницы

│ └── main_page.py # Главная страница

├── api/ # API клиент

│ └── kinopoisk_api.py # Класс для работы с API

├── tests/ # Тесты

│ ├── test_ui.py # UI тесты

│ └── test_api.py # API тесты

├── conftest.py # Фикстуры pytest

├── pytest.ini # Конфигурация pytest

├── requirements.txt # Зависимости

└── README.md # Документация



## Установка и запуск

### 1. Установка зависимостей
pip install -r requirements.txt

### 2. Запуск только UI тестов
pytest -m ui -v

### 3. Запуск только API тестов
pytest -m api -v

### 4. Запуск только smoke тестов
pytest -m smoke -v

### 5. Запуск всех тестов (без использования маркеров)
pytest -v

### 6. Запуск с отчетом Allure
pytest --alluredir=allure-results

allure serve allure-results
